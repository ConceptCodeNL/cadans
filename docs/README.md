# CADANS - Internship Management Platform Documentation

## Overview

CADANS is a student grading platform designed to streamline the evaluation and tracking of student performance through structured meeting-based grading, competency assessment, and end grade calculation. Built with Supabase and Vue.js, focusing on privacy (minimal personal data) and responsive mobile-first design.
